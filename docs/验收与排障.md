# 验收与排障清单

这份清单用于每次发新版本或部署到新电脑后快速检查。

## 1. 基础验收

### 设备加入

- 第一台电脑可以创建家庭网络。
- 设置页能复制加入密钥。
- 新电脑可以用加入密钥加入。
- 两台电脑都能在工作台看到对方。
- 设备显示名称、IP、平台、能力和在线状态。

### 设备整理

- 可以设置别名。
- 可以设置房间/分组。
- 可以收藏设备。
- 搜索设备可用。

## 2. 命令与终端

在本机执行：

```bash
lch devices --json
lch run --device 远端测试机 "hostname" --timeout-ms 30000 --json
lch run --device 远端测试机 "whoami" --timeout-ms 30000 --json
lch tasks --json
```

预期：

- 设备在线。
- 命令状态为 `completed`。
- 有 stdout。
- exitCode 为 `0`。

终端验收：

```bash
lch terminal 远端测试机
```

输入 `hostname`，应实时返回结果。

## 3. 看屏与远控

截图：

```bash
lch observe --device 远端测试机 --out screen.png --json
```

预期：

- 返回 width/height。
- 本地生成截图文件。
- 截图内容是远端真实桌面。

远控窗口：

```bash
lch remote open --device 远端测试机 --window --json
lch remote sessions --json
```

预期：

- 打开独立窗口。
- session 状态变成 `streaming` 或至少能刷新截图。
- 鼠标移动无明显堆积延迟。
- 点击位置基本对应远端画面。

## 4. 聊天与文件

聊天：

```bash
lch chat send --device 远端测试机 "hello"
```

文件：

```bash
lch file list --device 远端测试机
lch file send --device 远端测试机 "C:\Users\me\Desktop\note.txt"
```

预期：

- 消息出现在对话里。
- 文件发送记录出现。
- 共享目录能浏览和下载。

## 5. Windows 防火墙问题

现象：

- 设备列表显示在线。
- 但命令、截图、远控提示“远程设备响应超时”。

优先处理目标电脑：

```bash
lch firewall status --json
lch firewall repair
```

或在目标电脑 App 里：

```text
设置 -> Windows 防火墙 -> 修复防火墙
```

修复后重新测试：

```bash
lch run --device 远端测试机 "hostname" --timeout-ms 30000 --json
```

如果 `blockRules` 不是 `0`，说明 Windows 仍有阻止规则，需要在目标机允许权限确认。

## 6. 远控鼠标不准

可能原因：

- 远端 Windows 显示缩放较高。
- 远控窗口没有完整显示画面。
- 多显示器布局复杂。
- 实时视频和截图尺寸不同。

可尝试：

- 把远控窗口放大。
- 先用截图确认位置。
- 使用归一化坐标点击，例如 `--x 0.5 --y 0.5`。
- 中文输入使用剪贴板写入 + `Ctrl+V`。

后续建议增加“四角校准模式”，让用户点四个角后自动修正坐标偏差。

## 7. macOS 权限

macOS 首次使用看屏/控制需要允许：

- 屏幕录制。
- 辅助功能。

如果没有授权：

- 截图或实时流可能失败。
- 鼠标键盘注入可能失败。

授权后建议重启 App。


```text

```
